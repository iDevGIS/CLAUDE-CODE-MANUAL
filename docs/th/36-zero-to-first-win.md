# เริ่มจากศูนย์ จับมือทำ (Zero to First Win)

> **เป้าหมาย:** จากไม่รู้อะไรเลย → เห็นผลงานชิ้นแรกใน 15 นาที ทำตามทีละขั้น ไม่ต้องคิดเอง

หน้านี้เขียนให้ **คนที่ไม่เคยแตะคอมแบบเขียนโค้ดเลย** ถ้าคุณใช้ไลน์ ใช้เฟซ ซื้อของออนไลน์เป็น — คุณทำหน้านี้ได้แน่นอน 🙂

เราจะไม่อธิบายศัพท์ยากๆ ให้คุณ **อ่านทีละขั้น → ทำตามที่เห็น → กดตามที่บอก** แค่นั้น

มีตรงไหนหน้าตาไม่เหมือนในรูป? ไม่ต้องตกใจ เลื่อนไปอ่านกล่อง **💡 จุดที่มักงง** ที่อยู่ใกล้ๆ ขั้นนั้น

---

## ขั้นที่ 0: เช็คของก่อนเริ่ม

ก่อนอื่น มาดูว่าคุณมีครบไหม รายการนี้แค่ 2 อย่าง:

- ✅ **คอมพิวเตอร์** เครื่องไหนก็ได้ — Windows (Notebook/PC) หรือ Mac (MacBook/iMac)
- ✅ **อินเทอร์เน็ต** ต่อเน็ตได้ตามปกติ (ไวไฟบ้านก็พอ)

**ไม่ต้องมี:**
- ❌ ไม่ต้องเก่งคอม
- ❌ ไม่ต้องเคยเขียนโปรแกรม
- ❌ ไม่ต้องรู้ภาษาอังกฤษเป๊ะ (พิมพ์ไทยสั่งงานได้)

> 💡 **จุดที่มักงง: "เครื่องผมจะไหวไหม?"**
> ไหวครับ Claude Code ทำงานบนเซิร์ฟเวอร์ของ Anthropic ไม่ได้หนักเครื่องคุณ ขอแค่เครื่องเปิดเน็ตได้ก็พอ คอมเก่าๆ อายุ 5-6 ปีก็ใช้ได้

พร้อมแล้ว ไปกันเลย 👇

---

## ขั้นที่ 1: ติดตั้ง Node.js (โปรแกรมพื้นฐานที่ Claude ต้องใช้)

Claude Code ต้องการตัวช่วยเล็กๆ ชื่อ **Node.js** ก่อน (ลงครั้งเดียวจบ ไม่ต้องลงซ้ำ) ทำตามนี้:

1. เปิดเว็บเบราว์เซอร์ (Chrome / Edge / Safari อันไหนก็ได้)
2. พิมพ์ที่อยู่เว็บนี้ลงไป แล้วกด Enter:

```
nodejs.org
```

3. หน้าเว็บจะมี **ปุ่มเขียวๆ ใหญ่ๆ** ตรงกลาง เขียนว่า **"LTS"** — กดปุ่มนั้น (LTS = รุ่นที่นิ่งที่สุด แนะนำสำหรับทุกคน)
4. ไฟล์จะดาวน์โหลดลงเครื่อง รอจนเสร็จ แล้ว **ดับเบิลคลิก** เปิดไฟล์นั้น
5. หน้าต่างติดตั้งจะเด้งขึ้นมา — **กด "Next" (ถัดไป) ไปเรื่อยๆ** ไม่ต้องไปติ๊กอะไรเพิ่ม เจอ "Agree/ยอมรับ" ก็กดยอมรับ สุดท้ายกด **"Install" / "Finish"**

👀 **สิ่งที่จะเห็นบนจอ (หน้าเว็บ nodejs.org):**

```
┌─────────────────────────────────────────────┐
│              Node.js                         │
│                                              │
│   Download Node.js (LTS)   ← กดปุ่มเขียวนี้    │
│   [  ⬇  v22.x.x  LTS  ]                       │
│                                              │
│   Recommended for most users                 │
└─────────────────────────────────────────────┘
```

### เช็คว่าลงสำเร็จไหม

หลังลงเสร็จ เราจะเช็คว่าใช้ได้จริง ขั้นนี้ต้องเปิด **หน้าต่างพิมพ์คำสั่ง** ก่อน:

- **Windows:** กดปุ่ม Start (โลโก้ Windows มุมล่างซ้าย) → พิมพ์คำว่า `PowerShell` → กดเปิด **Windows PowerShell**
- **Mac:** กด `Command (⌘) + Spacebar` พร้อมกัน → พิมพ์คำว่า `Terminal` → กด Enter

จะได้หน้าต่างดำๆ (หรือน้ำเงิน) ขึ้นมา พิมพ์บรรทัดนี้ลงไปแล้วกด Enter:

```bash
node -v
```

👀 **สิ่งที่จะเห็นบนจอ:**

```
PS C:\Users\YourName> node -v
v22.11.0
```

ถ้าขึ้นเลขแบบ `v22.x.x` (เลขอาจไม่ตรงเป๊ะ ไม่เป็นไร) = **สำเร็จ!** ✅

> 💡 **จุดที่มักงง: ขึ้นว่า `node : command not found` หรือ `ไม่รู้จักคำสั่ง node`**
> แปลว่า Node.js ยังลงไม่สมบูรณ์ หรือคุณยังไม่ได้ **ปิดแล้วเปิดหน้าต่าง PowerShell/Terminal ใหม่** วิธีแก้: ปิดหน้าต่างดำๆ นั้นไปก่อน แล้วเปิดขึ้นมาใหม่ แล้วพิมพ์ `node -v` อีกที ถ้ายังไม่ได้ ลองติดตั้ง Node.js ใหม่ตามขั้นที่ 1 อีกรอบ

## ขั้นที่ 2: ติดตั้ง Claude Code

ตอนนี้เรามี Node.js แล้ว มาลง **Claude Code** กัน ใน **หน้าต่างเดิม** (PowerShell บน Windows / Terminal บน Mac) พิมพ์บรรทัดยาวๆ นี้ลงไป แล้วกด Enter:

```bash
npm install -g @anthropic-ai/claude-code
```

> เคล็ดลับ: พิมพ์ยาวกลัวผิด? ให้ **copy ทั้งบรรทัด** จากหน้านี้ แล้วเอาไปวางในหน้าต่างนั้น (Windows คลิกขวาเพื่อวาง / Mac กด `⌘ + V`)

รอสักครู่ (10-60 วินาที แล้วแต่ความเร็วเน็ต) จะมีตัวหนังสือวิ่งๆ เต็มจอ — **ปกติ ไม่ต้องตกใจ** รอจนมันหยุดและกลับมาที่บรรทัดให้พิมพ์

👀 **สิ่งที่จะเห็นบนจอ:**

```
PS C:\Users\YourName> npm install -g @anthropic-ai/claude-code

added 254 packages in 23s

PS C:\Users\YourName> _
```

เห็นคำว่า `added ... packages` และเคอร์เซอร์กลับมากระพริบที่บรรทัดใหม่ = **ลงเสร็จแล้ว!** ✅

> 💡 **จุดที่มักงง: ขึ้นข้อความสีแดง / สีเหลืองเต็มไปหมด**
> ข้อความสีเหลือง (warning) ส่วนใหญ่ **ข้ามได้** ไม่ใช่ปัญหา แต่ถ้าเป็นสีแดงและจบด้วยคำว่า `permission denied` หรือ `EACCES`:
> - **Mac:** พิมพ์แบบนี้แทน → `sudo npm install -g @anthropic-ai/claude-code` แล้วมันจะถามรหัสผ่านเครื่อง (พิมพ์ไปเลย จอจะไม่ขึ้นดาว ๆ เป็นเรื่องปกติ) กด Enter
> - **Windows:** ปิด PowerShell แล้วเปิดใหม่แบบ **"Run as Administrator"** (คลิกขวาที่ไอคอน PowerShell → Run as administrator) แล้วลองพิมพ์คำสั่งเดิมอีกที

---

## ขั้นที่ 3: เปิด Claude Code ครั้งแรก

ก่อนเปิด เราควรมี **โฟลเดอร์** ไว้เก็บงาน (เหมือนสร้างแฟ้มเปล่าๆ ไว้ใส่เอกสาร) ใน **หน้าต่างเดิม** พิมพ์ทีละบรรทัด (พิมพ์บรรทัดแรก กด Enter แล้วค่อยพิมพ์บรรทัดถัดไป):

```bash
mkdir my-first-claude
cd my-first-claude
```

- `mkdir my-first-claude` = สร้างโฟลเดอร์ชื่อ `my-first-claude`
- `cd my-first-claude` = เดินเข้าไปในโฟลเดอร์นั้น (cd ย่อมาจาก "change directory")

จากนั้นพิมพ์คำวิเศษ แล้วกด Enter:

```bash
claude
```

👀 **สิ่งที่จะเห็นบนจอ (ครั้งแรกสุด):**

```
┌────────────────────────────────────────────┐
│  ✻ Welcome to Claude Code                    │
│                                              │
│  Let's get you logged in.                    │
│                                              │
│  Open this URL in your browser to login:     │
│  https://claude.ai/oauth/authorize?...       │
│                                              │
│  Waiting for authentication...               │
└────────────────────────────────────────────┘
```

เห็นข้อความชวน login พร้อมลิงก์ยาวๆ ใช่ไหม? นั่นแหละ ไปขั้นต่อไป 👇

> 💡 **จุดที่มักงง: พิมพ์ `claude` แล้วขึ้นว่า `command not found`**
> ปิดหน้าต่างดำๆ แล้วเปิดขึ้นมาใหม่อีกครั้ง (เครื่องต้อง "รับรู้" โปรแกรมใหม่ที่เพิ่งลง) แล้วพิมพ์ `cd my-first-claude` ตามด้วย `claude` อีกที ส่วนใหญ่หายเลย

---

## ขั้นที่ 4: Login (เข้าสู่ระบบ)

ครั้งแรกที่เปิด Claude จะให้คุณ login เพื่อยืนยันว่าเป็นคุณจริงๆ ทำตามนี้:

1. ในหน้าต่างมีลิงก์ยาวๆ ขึ้นว่า `https://claude.ai/oauth/...`
   - **กดลิงก์นั้นได้เลย** (บางเครื่อง: กดปุ่ม `Ctrl` ค้างแล้วคลิก / บน Mac กด `⌘` ค้างแล้วคลิก)
   - ถ้าคลิกไม่ได้ ให้ลาก **copy ลิงก์ทั้งเส้น** ไปวางในช่องที่อยู่ของเบราว์เซอร์เอง แล้ว Enter
2. เบราว์เซอร์จะเปิดหน้า login ของ Claude — **เข้าด้วยอีเมล** (หรือกดปุ่ม "Continue with Google" ถ้าใช้ Gmail จะง่ายสุด)
3. พอ login เสร็จ เว็บจะขึ้นปุ่ม **"Authorize" / "อนุญาต"** — กดเลย
4. หน้าเว็บจะบอกว่าสำเร็จ ให้ **กลับไปที่หน้าต่างดำๆ (PowerShell/Terminal)** ของเรา

👀 **สิ่งที่จะเห็นบนจอ (กลับมาที่หน้าต่างคำสั่ง):**

```
┌────────────────────────────────────────────┐
│  ✓ Login successful!                          │
│                                              │
│  ✻ Claude Code                                │
│                                              │
│  > _                                          │
│    พิมพ์อะไรก็ได้ตรงนี้...                       │
└────────────────────────────────────────────┘
```

เห็นเครื่องหมาย `>` กับเคอร์เซอร์กระพริบ = **พร้อมรับคำสั่งแล้ว!** 🎉

> 💡 **จุดที่มักงง: เบราว์เซอร์ไม่เด้งขึ้นมาเอง / ไม่ขึ้นหน้า login**
> ไม่เป็นไร ให้ copy ลิงก์ยาวๆ ในหน้าต่างคำสั่ง (ลากดำทั้งเส้น → คลิกขวา copy) แล้วเปิดเบราว์เซอร์เอง วางลิงก์ในช่องที่อยู่ด้านบน กด Enter ก็เข้าหน้า login ได้เหมือนกัน

## ขั้นที่ 5: งานแรกที่ได้ "win" — สั่งเป็นภาษาคน

ถึงเวลาสนุกแล้ว! ตอนนี้เคอร์เซอร์กระพริบหลัง `>` ให้ **พิมพ์เป็นภาษาไทยปกติ** เหมือนสั่งผู้ช่วย ลองพิมพ์ประโยคนี้แล้วกด Enter:

```bash
ช่วยสร้างไฟล์ hello.txt เขียนว่า "สวัสดีจาก Claude" ให้หน่อย
```

Claude จะอ่านที่คุณพิมพ์ คิดสักครู่ แล้วบอกว่ามันกำลังจะสร้างไฟล์ให้

👀 **สิ่งที่จะเห็นบนจอ:**

```
> ช่วยสร้างไฟล์ hello.txt เขียนว่า "สวัสดีจาก Claude" ให้หน่อย

  ได้เลยครับ ผมจะสร้างไฟล์ hello.txt ให้

  ● Write(hello.txt)
    เนื้อหา: สวัสดีจาก Claude
```

ตรงนี้มันจะ **หยุดรอ** ถามคุณก่อนว่า "จะให้สร้างไฟล์จริงไหม?" — ไปขั้นต่อไป 👇

> 💡 **จุดที่มักงง: พิมพ์ไทยได้ไหม?**
> ได้ 100%! Claude เข้าใจภาษาไทยดีมาก สั่งงานเป็นไทยได้เลย ไม่ต้องแปลเป็นอังกฤษ ถ้าพิมพ์ไทยแล้วตัวอักษรเพี้ยน ลองเปลี่ยนหน้าต่างเป็น Windows Terminal (รุ่นใหม่) หรือใช้ Terminal ของ Mac ซึ่งรองรับไทยเต็มที่

## ขั้นที่ 6: ตอบกล่องขออนุญาต (Permission)

เพื่อความปลอดภัย Claude จะ **ถามก่อนเสมอ** เวลาจะแก้ไฟล์ในเครื่องคุณ — เป็นเรื่องดี! คุณคุมได้ตลอด

จะมีกล่องเด้งขึ้นมาถามทำนองว่า "ขออนุญาตสร้าง/แก้ไฟล์นี้ไหม?" พร้อมตัวเลือก

👀 **สิ่งที่จะเห็นบนจอ:**

```
┌────────────────────────────────────────────┐
│  Claude wants to create hello.txt            │
│                                              │
│  สวัสดีจาก Claude                              │
│                                              │
│  Do you want to proceed?                     │
│  ❯ 1. Yes                                     │
│    2. Yes, and don't ask again               │
│    3. No, tell Claude what to do differently  │
└────────────────────────────────────────────┘
```

ให้เลือก **ข้อ 1 (Yes)** ทำได้ 2 วิธี:
- กดเลข `1` แล้ว Enter **หรือ**
- ใช้ปุ่มลูกศรขึ้น/ลงเลื่อนให้ลูกศร `❯` อยู่หน้า "Yes" แล้วกด Enter

แค่นี้ Claude ก็จะลงมือสร้างไฟล์ให้จริง ✅

> 💡 **จุดที่มักงง: เลือกข้ออะไรดี?**
> - **ข้อ 1 (Yes)** = อนุญาตครั้งนี้ครั้งเดียว — ปลอดภัยสุด แนะนำสำหรับมือใหม่
> - **ข้อ 2 (Yes, and don't ask again)** = อนุญาตและไม่ต้องถามซ้ำ — สะดวกขึ้นแต่เผลอกดงานอื่นได้ ค่อยใช้ตอนชินแล้ว
> - **ข้อ 3 (No)** = ไม่เอา แล้วบอก Claude ว่าอยากให้ทำอะไรแทน
> มือใหม่: **กด 1 ไปก่อนทุกครั้ง** ก็พอครับ

## ขั้นที่ 7: เห็นผล + ลองเปิดไฟล์ดู 🎉

Claude จะบอกว่าทำเสร็จแล้ว

👀 **สิ่งที่จะเห็นบนจอ:**

```
  ● Write(hello.txt) ✓
    สร้างไฟล์ hello.txt เรียบร้อยแล้วครับ ✅

> _
```

**มาเปิดไฟล์ดูของจริงกัน!** เปิดได้ 2 ทาง เลือกอันที่ถนัด:

**ทางที่ 1 — ให้ Claude เปิดให้:** พิมพ์ในหน้าต่างเดิมแล้ว Enter

```bash
ขอดูเนื้อหาในไฟล์ hello.txt หน่อย
```

Claude จะอ่านไฟล์แล้วโชว์ข้อความ `สวัสดีจาก Claude` ให้คุณเห็นตรงนั้นเลย

**ทางที่ 2 — เปิดด้วยตาตัวเอง:** เปิดโปรแกรมจัดการไฟล์ (Windows = File Explorer / Mac = Finder) แล้วเข้าไปที่โฟลเดอร์ `my-first-claude` (มักอยู่ใน C:\Users\ชื่อคุณ\ บน Windows หรือ Home บน Mac) จะเห็นไฟล์ **hello.txt** นอนรออยู่ — ดับเบิลคลิกเปิด จะเห็นข้อความที่ Claude เขียนให้

🎉🎉 **ยินดีด้วย! นี่คือผลงานชิ้นแรกของคุณกับ Claude Code!** 🎉🎉

คุณเพิ่งสั่งงาน AI ด้วยภาษาคน แล้วมันสร้างไฟล์จริงในเครื่องคุณได้สำเร็จ จากที่เมื่อ 15 นาทีก่อนยังไม่รู้อะไรเลย เก่งมาก! 👏

ถ้าเริ่มสนุกแล้ว ลองสั่งงานอื่นๆ ต่อได้เลย เช่น:

- `ช่วยสร้างไฟล์ todo.txt ใส่รายการของที่ต้องซื้อ 5 อย่างให้หน่อย`
- `ในโฟลเดอร์นี้มีไฟล์อะไรบ้าง`
- `ช่วยแก้ไฟล์ hello.txt เพิ่มบรรทัดใหม่ว่า "วันนี้อากาศดี" ให้หน่อย`

อยากออกจาก Claude เมื่อไหร่? พิมพ์ `/exit` แล้ว Enter (หรือกด `Ctrl + C` สองครั้ง)
