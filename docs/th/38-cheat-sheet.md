# การ์ดสรุปฉบับพกพา (Cheat Sheet)

> **เป้าหมาย:** ทุกอย่างที่มือใหม่ต้องใช้ รวมในหน้าเดียว — ปริ้นต์แปะข้างจอได้เลย

## 🚀 เริ่มงาน 4 ขั้น

| ขั้น | ทำอะไร | พิมพ์ |
|:--:|--------|------|
| 1️⃣ | เข้าโฟลเดอร์งานก่อน | `cd โฟลเดอร์ของฉัน` |
| 2️⃣ | เปิด Claude | `claude` |
| 3️⃣ | สั่งงานเป็น **ภาษาคน** | `ช่วยอ่านไฟล์ในโฟลเดอร์นี้ให้หน่อย` |
| 4️⃣ | มันขออนุญาต → กดตกลง | `y` แล้ว `Enter` |

> 💡 จำแค่ 3 ตัวพอ: `cd` → `claude` → พิมพ์งาน แล้วตอบ `y`

## 💻 คำสั่งเปิดโปรแกรม

| พิมพ์ในเทอร์มินัล | ได้อะไร |
|------|--------|
| `claude` | เริ่มงานใหม่ (คุยกันไปเรื่อย ๆ) |
| `claude -c` | ทำต่อของเดิม — มันจำเรื่องที่คุยค้างไว้ |
| `claude -p "คำถาม"` | ถามสั้น ๆ ตอบเสร็จแล้วจบเลย ไม่ค้างหน้าจอ |
| `claude update` | อัปเดตเป็นเวอร์ชันล่าสุด |

```bash
claude -p "สรุปไฟล์ README ให้ 2 บรรทัด"
```

## ⚡ Slash ที่ใช้บ่อย

> พิมพ์ `/` หน้าบรรทัดเพื่อเรียกเมนูคำสั่ง

| คำสั่ง | ทำอะไร |
|------|--------|
| `/help` | ขอความช่วยเหลือ / ดูคำสั่งทั้งหมด |
| `/clear` | ล้างความจำ เริ่มคุยใหม่ตั้งแต่ต้น |
| `/compact` | คุยยาวจนใกล้เต็ม → ย่อให้สั้นลงเพื่อคุยต่อ |
| `/model` | สลับสมอง (Opus / Sonnet / Haiku) |
| `/rewind` | ย้อนเวลา กลับไปจุดก่อนหน้า (โค้ด + บทคุย) |
| `/cost` | ดูว่าใช้เงินไปเท่าไหร่แล้ว |
| `/config` | เปิดหน้าตั้งค่า |
| `/exit` | ออกจากโปรแกรม |

## 🎹 คีย์ลัดที่ต้องรู้

| กด | ได้อะไร |
|------|--------|
| `Ctrl + C` | **หยุด!** มันทำงานอยู่ ให้เบรกทันที |
| `Esc` `Esc` | ย้อนกลับ (กด Esc สองที) เปิดเมนูถอยเวลา |
| `Shift + Tab` | สลับโหมดความปลอดภัย (เช่น เข้า Plan Mode) |
| `Ctrl + D` | ออกจากโปรแกรม |
| `@` | แนบไฟล์ (พิมพ์ `@` แล้วชื่อไฟล์เด้งขึ้นมา) |
| `!` | รันคำสั่ง bash ตรง ๆ (เช่น `!ls`) |

## 💬 "อยากให้... พิมพ์แบบนี้"

> สั่งงานเป็นภาษาคนได้เลย ไม่ต้องท่องคำสั่ง

| อยากได้ | พิมพ์แบบนี้ | มันจะ... |
|--------|-----------|---------|
| 📖 เข้าใจโค้ด | `อธิบายไฟล์ app.py ให้ฟังหน่อย` | อ่านแล้วเล่าให้ฟังเป็นภาษาคน |
| 🐞 แก้บั๊ก | `รันแล้ว error แบบนี้ ช่วยแก้ที` | หาต้นเหตุแล้วแก้ให้ |
| 📄 สร้างไฟล์ | `สร้างไฟล์ index.html หน้าแรกง่าย ๆ` | เขียนไฟล์ใหม่ให้ |
| 🧪 เขียนเทสต์ | `เขียน test ให้ฟังก์ชันนี้หน่อย` | สร้างไฟล์เทสต์ครอบเคส |
| 📝 สรุป git | `สรุปว่า 10 commit ล่าสุดทำอะไรบ้าง` | อ่าน git log แล้วสรุปให้ |
| 🌐 แปลภาษา | `แปลคอมเมนต์ในไฟล์นี้เป็นไทย` | แก้คอมเมนต์ให้เป็นไทย |

## 🛟 ฉุกเฉิน

| อาการ | ทำอะไร |
|------|--------|
| 😱 มันทำผิด อยากถอย | กด `Esc` `Esc` |
| 🧊 ค้าง / ช้า / วน | กด `Ctrl + C` |
| 😵 งง ไม่รู้จะทำไง | พิมพ์ `/help` |
| 😨 กลัวมันพังของ | เข้า **Plan Mode** ก่อน (`Shift + Tab`) — ให้มันวางแผนให้ดูก่อนลงมือ |

> ✅ **กฎเหล็ก:** ไม่แน่ใจอะไร ให้กด `Esc` `Esc` ไว้ก่อน — ปลอดภัยสุด
