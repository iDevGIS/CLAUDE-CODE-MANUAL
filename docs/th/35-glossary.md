# อภิธานศัพท์ฉบับบ้าน ๆ (Glossary)

> **เป้าหมาย:** แปลศัพท์ที่เจอบ่อยให้เป็นภาษาคน — อ่านแล้วเข้าใจทันที ไม่ต้องเป็นโปรแกรมเมอร์

เจอศัพท์แปลก ๆ ในคัมภีร์เล่มนี้แล้วงงใช่ไหม? ไม่ต้องกลัว ตารางข้างล่างแปลให้หมดแบบบ้าน ๆ เทียบกับของในบ้านที่คุณคุ้นเคย อ่านทีละช่องได้เลย

## 🏠 พื้นฐานสุด ๆ (เริ่มจากตรงนี้)

| คำศัพท์ | แปลบ้าน ๆ | เทียบของในบ้าน |
|---------|-----------|----------------|
| **Terminal** | ช่องดำ ๆ ไว้พิมพ์คำสั่งให้คอม | ช่องส่งโพยให้แม่ครัวในร้านอาหาร |
| **Command line (CLI)** | การสั่งคอมด้วยการ "พิมพ์" แทนการ "คลิก" | สั่งกาแฟด้วยปากแทนกดตู้กดเอง |
| **Command** | คำสั่ง 1 บรรทัดที่บอกให้คอมทำอะไร | "ปิดไฟห้องครัวที" — สั่งทีนึงทำทีนึง |
| **Prompt** | ข้อความที่เราพิมพ์บอก AI ว่าจะให้ทำอะไร | โพยสั่งอาหารที่เขียนรายละเอียดให้ครบ |
| **Enter** | ปุ่มกด "ส่ง" คำสั่งให้เริ่มทำงาน | กดกริ่งเรียกให้พนักงานมารับโพย |
| **Directory / Folder** | กล่องเก็บไฟล์ ไว้จัดของให้เป็นหมวด | ลิ้นชักในตู้ — แยกช้อนส้อม แยกจาน |
| **Path** | ที่อยู่ของไฟล์ บอกว่าของอยู่ลิ้นชักไหน | ที่อยู่บ้าน "ซอย 5 บ้านเลขที่ 3 ห้องครัว" |
| **Install** | ลงโปรแกรมเข้าเครื่องให้พร้อมใช้ | ติดตั้งเตาแก๊สใหม่ก่อนเริ่มทำกับข้าว |

## 🤖 AI กับโมเดล (สมองของ Claude)

| คำศัพท์ | แปลบ้าน ๆ | เทียบของในบ้าน |
|---------|-----------|----------------|
| **AI model** | "สมอง" ปัญญาประดิษฐ์ที่คิดและตอบให้ | พ่อครัวคนหนึ่งที่มีฝีมือต่างกันไป |
| **Opus** | สมองรุ่นเก่งสุด คิดลึก งานยากใหญ่ ๆ | เชฟใหญ่ฝีมือเทพ แต่คิวยาวหน่อย |
| **Sonnet** | สมองรุ่นกลาง เก่ง+เร็ว สมดุลดี | กุ๊กมือดีประจำร้าน ใช้งานได้ทุกวัน |
| **Haiku** | สมองรุ่นเล็ก เร็วมาก งานเบา ๆ | เด็กในครัวที่ว่องไว ทำงานง่าย ๆ ฉับ ๆ |
| **Fable 5** | สมองรุ่นใหม่ เน้นความเร็วและงานทั่วไป | กุ๊กรุ่นใหม่ไฟแรง คล่องตัว |
| **Token** | หน่วยนับคำ ใช้คิดราคาและความยาว | คิดเงินเป็น "คำ" เหมือนค่าโทรคิดเป็นนาที |
| **Context (window)** | "ความจำชั่วคราว" ของ AI ในรอบคุยนี้ | โต๊ะทำงานที่วางของได้จำกัด เต็มแล้วต้องเก็บ |
| **Extended thinking** | โหมดคิดนาน ๆ ก่อนตอบ เพื่องานยาก | เชฟหยุดคิดสูตรให้รอบคอบก่อนลงมือ |
| **Fast mode** | โหมดตอบไว เน้นเร็วกว่าคิดลึก | สั่งจานด่วน เอาเร็วไว้ก่อน |

## 📄 โค้ดกับไฟล์ (ของที่ AI เข้าไปจัดการ)

| คำศัพท์ | แปลบ้าน ๆ | เทียบของในบ้าน |
|---------|-----------|----------------|
| **Code** | ภาษาที่ใช้สั่งให้คอมทำงาน | สูตรอาหารที่เขียนเป็นขั้นตอนละเอียด |
| **Repo (repository)** | กล่องใหญ่เก็บไฟล์ทั้งโปรเจกต์ | ตู้เก็บสูตรอาหารทั้งเล่มของร้าน |
| **Git** | โปรแกรมจดบันทึก "ทุกการแก้ไข" ของไฟล์ | สมุดบันทึกว่าวันไหนเปลี่ยนสูตรอะไร |
| **GitHub** | เว็บเก็บโค้ดออนไลน์ ไว้แชร์/สำรอง | ตู้เซฟกลางบนคลาวด์ ที่ทีมเปิดดูร่วมกัน |
| **Commit** | การ "บันทึกจุดเซฟ" ของงานที่แก้ | ถ่ายรูปจานที่ทำเสร็จเก็บไว้เป็นหลักฐาน |
| **Push / Pull** | ส่งงานขึ้นเว็บ / ดึงงานล่าสุดลงมา | ส่งของเข้าตู้เซฟกลาง / หยิบของจากตู้มาใช้ |
| **Branch** | สายงานแยก ไว้ลองของใหม่ไม่ให้กระทบของเดิม | ทำสูตรทดลองในหม้อแยก ไม่ปนหม้อหลัก |
| **Node.js** | โปรแกรมพื้นฐานที่ Claude Code ต้องใช้รัน | เตาแก๊ส — ต้องมีก่อนถึงจะทำกับข้าวได้ |
| **npm** | ตัวช่วยดาวน์โหลด/ติดตั้งโปรแกรมเสริม | ซูเปอร์มาร์เก็ตสั่งวัตถุดิบมาเข้าครัว |
| **Terminal vs PowerShell** | ช่องพิมพ์คำสั่ง 2 แบบ (Mac/Linux กับ Windows) | เตาแก๊ส กับ เตาไฟฟ้า — คนละยี่ห้อ ทำกับข้าวได้เหมือนกัน |

## ✨ ฟีเจอร์ Claude Code (ลูกเล่นเฉพาะตัว)

| คำศัพท์ | แปลบ้าน ๆ | เทียบของในบ้าน |
|---------|-----------|----------------|
| **CLAUDE.md** | ไฟล์โน้ตบอกกฎ/ข้อมูลโปรเจกต์ให้ Claude อ่านก่อนเริ่ม | โพยติดตู้เย็น "บ้านนี้กินเผ็ดน้อย แพ้กุ้ง" |
| **Slash command (/)** | คำสั่งลัดพิมพ์ขึ้นต้นด้วย / เรียกใช้ง่าย ๆ | ปุ่มลัดบนรีโมต กดทีเดียวทำงานเลย |
| **Flag (--)** | ตัวเลือกเสริมต่อท้ายคำสั่ง ปรับการทำงาน | สั่ง "เพิ่มพิเศษ" ต่อท้าย เช่น "ไม่ใส่ผัก" |
| **@ file** | พิมพ์ @ แล้วชื่อไฟล์ เพื่อชี้ให้ Claude ดูไฟล์นั้น | ชี้นิ้วบอก "เอาจานนี้แหละ" ให้ดูตรง ๆ |
| **! bash mode** | พิมพ์ ! นำหน้าเพื่อสั่งคอมตรง ๆ ไม่ผ่าน AI | กดสวิตช์เองโดยไม่ต้องเรียกพนักงาน |
| **Permission** | การ "ขออนุญาต" ก่อน Claude ลงมือทำของจริง | พนักงานถาม "ปรุงเลยไหมครับ?" ก่อนลงมือ |
| **Plan Mode** | โหมดให้ Claude "วางแผนก่อน" ยังไม่แตะของจริง | เชฟร่างเมนูให้ดูก่อน ค่อยตกลงแล้วเริ่มทำ |
| **Hook** | กับดักอัตโนมัติ สั่งให้ทำบางอย่างทุกครั้งที่... | ตั้งเวลาให้เครื่องล้างจานทำงานเองหลังกินเสร็จ |
| **Skill** | ชุดความรู้/ทักษะพิเศษที่สอน Claude เพิ่มได้ | คอร์สสอนทำขนมที่เชฟไปเรียนมาเสริม |
| **Subagent** | ผู้ช่วยตัวเล็กที่ Claude แยกไปทำงานย่อย | ลูกมือที่หัวหน้าเชฟส่งไปหั่นผักให้เสร็จ |
| **Agent Team** | ทีมผู้ช่วยหลายตัวช่วยกันทำงานใหญ่พร้อมกัน | ทีมครัวหลายคนช่วยกันทำงานเลี้ยงโต๊ะจีน |
| **MCP** | ท่อต่อให้ Claude คุยกับโปรแกรม/บริการอื่นได้ | สายต่อให้ครัวสั่งของจากร้านข้าง ๆ ได้เลย |
| **Plugin** | ของเสริมที่ติดตั้งเพิ่มเพื่อเพิ่มความสามารถ | อุปกรณ์เสริมเตา เช่น ที่ปิ้งย่างต่อเพิ่ม |
| **Headless** | สั่ง Claude ทำงานเองเงียบ ๆ ไม่ต้องนั่งเฝ้า | ตั้งหม้อหุงข้าวไว้ แล้วเดินไปทำอย่างอื่น |
| **Sandbox** | พื้นที่ปลอดภัยให้ลองทำโดยไม่กระทบของจริง | ครัวซ้อมที่พังได้ ไม่เสียดายของจริง |
| **Worktree** | สำเนาโปรเจกต์แยกไว้ทำงานหลายอย่างพร้อมกัน | โต๊ะเตรียมอาหารตัวที่สอง ไว้ทำคนละเมนู |
| **Session** | ช่วงการคุย 1 รอบกับ Claude ตั้งแต่เปิดถึงปิด | การนั่งกินข้าว 1 มื้อ ตั้งแต่สั่งจนอิ่ม |
| **Memory** | ความจำถาวรที่ Claude จำข้ามรอบคุยได้ | สมุดจดที่เชฟเปิดดูได้ทุกวัน ไม่ลืม |
