# Glossary — Plain-Language Terms

> **Goal:** Translate the jargon you'll meet into plain language — understand it instantly, no programming required

Bumped into a strange word in this manual and felt lost? Don't worry. The tables below decode every term in plain language and compare it to something familiar from around the house. Just read one cell at a time.

## 🏠 Absolute basics (start here)

| Term | Plain meaning | Everyday analogy |
|------|---------------|------------------|
| **Terminal** | A black box where you type commands to the computer | The slip you hand the kitchen to order food |
| **Command line (CLI)** | Telling the computer by "typing" instead of "clicking" | Ordering coffee by speaking instead of using a machine |
| **Command** | One line that tells the computer to do one thing | "Turn off the kitchen light" — one order, one action |
| **Prompt** | The message you type telling the AI what to do | An order slip with all the details written out |
| **Enter** | The key that "sends" the command to start it | Ringing the bell so staff come pick up your slip |
| **Directory / Folder** | A box that holds files, to keep things sorted | A drawer in a cabinet — spoons here, plates there |
| **Path** | A file's address, saying which drawer it's in | A home address: "Lane 5, House 3, the kitchen" |
| **Install** | Putting a program onto your machine, ready to use | Fitting a new gas stove before you start cooking |

## 🤖 AI & models (Claude's brains)

| Term | Plain meaning | Everyday analogy |
|------|---------------|------------------|
| **AI model** | The artificial "brain" that thinks and answers | A cook — each one has a different skill level |
| **Opus** | The smartest model, deep thinker, big hard jobs | The head chef — brilliant, but the line is longer |
| **Sonnet** | The mid model, smart + fast, nicely balanced | The reliable line cook you use every day |
| **Haiku** | The small model, very fast, light jobs | The quick kitchen hand who zips through simple tasks |
| **Fable 5** | A newer model tuned for speed and everyday work | A nimble new cook with lots of energy |
| **Token** | The unit that counts words, used for price and length | Charging by the "word," like a call billed per minute |
| **Context (window)** | The AI's "short-term memory" for this chat | A work desk with limited space — when full, clear it |
| **Extended thinking** | A mode where it thinks longer before answering | The chef pausing to plan the recipe carefully first |
| **Fast mode** | A mode that answers quickly, speed over depth | Ordering an express dish — quick first, fancy later |

## 📄 Code & files (what the AI works on)

| Term | Plain meaning | Everyday analogy |
|------|---------------|------------------|
| **Code** | The language used to tell a computer what to do | A recipe written as detailed step-by-step instructions |
| **Repo (repository)** | The big box holding all of a project's files | The cabinet storing the restaurant's whole recipe book |
| **Git** | A program that records "every change" to your files | A logbook noting which day you changed which recipe |
| **GitHub** | A website that stores code online, to share/back up | A shared cloud safe the whole team can open together |
| **Commit** | "Saving a checkpoint" of the work you changed | Photographing a finished dish to keep as a record |
| **Push / Pull** | Send your work up online / fetch the latest down | Putting items in the shared safe / taking them out to use |
| **Branch** | A side track for trying new things safely | Testing a recipe in a separate pot, not the main one |
| **Node.js** | The base program Claude Code needs in order to run | The gas stove — you need it before you can cook |
| **npm** | A helper that downloads/installs add-on programs | A supermarket that delivers ingredients to your kitchen |
| **Terminal vs PowerShell** | Two flavors of command box (Mac/Linux vs Windows) | A gas stove vs an electric one — different brand, same cooking |

## ✨ Claude Code features (its special tricks)

| Term | Plain meaning | Everyday analogy |
|------|---------------|------------------|
| **CLAUDE.md** | A note file of project rules/info Claude reads first | The fridge note: "This house likes mild, allergic to shrimp" |
| **Slash command (/)** | A shortcut command that starts with / for quick use | A shortcut button on the remote — one press, done |
| **Flag (--)** | An extra option added to a command to tweak it | Adding "extra requests" like "hold the veggies" |
| **@ file** | Type @ then a filename to point Claude at that file | Pointing and saying "this dish right here" |
| **! bash mode** | Type ! first to run a computer command directly | Flipping the switch yourself, no staff needed |
| **Permission** | The "asking first" before Claude does the real thing | Staff asking "Shall I cook it now?" before starting |
| **Plan Mode** | A mode where Claude "plans first," touches nothing yet | The chef sketching the menu for you to approve first |
| **Hook** | An automatic trigger that runs something every time... | The dishwasher set to run on its own after you eat |
| **Skill** | A bundle of special know-how you can teach Claude | A pastry course the chef took to add to their skills |
| **Subagent** | A little helper Claude splits off for a small subtask | The sous-hand the head chef sends to chop the veggies |
| **Agent Team** | Several helpers working a big job together at once | A full kitchen crew handling a big banquet together |
| **MCP** | A pipe that lets Claude talk to other programs/services | A line letting your kitchen order from the shop next door |
| **Plugin** | An add-on you install to give Claude more abilities | A stove attachment, like a grill plate you bolt on |
| **Headless** | Telling Claude to work on its own, no babysitting | Setting the rice cooker and walking off to do other things |
| **Sandbox** | A safe space to try things without touching the real one | A practice kitchen you can wreck — no real loss |
| **Worktree** | A separate copy of the project for parallel work | A second prep table for cooking a different menu |
| **Session** | One round of chatting with Claude, open to close | One sit-down meal, from ordering until you're full |
| **Memory** | Permanent memory Claude keeps across chats | A notebook the chef reads every day and never forgets |
